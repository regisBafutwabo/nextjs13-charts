# Clika-Charts

## Getting Started

First, install the dependencies:
```bash

npm i
# or
yarn

```

Then, run the development server:

```bash

npm run dev
# or
yarn dev

```
## Dependencies

This Project is using:
- Nextjs 13
- Tailwind-css
- Recharts
